---

## 🌍 Cómo contribuir

### 1. Reportar errores o sugerencias
Abre un [Issue](https://github.com/juanmoisesd/indicadores-salud-mental-latinoamerica/issues) describiendo:
- El problema encontrado o la mejora propuesta
- Pasos para reproducir el error (si aplica)
- Capturas de pantalla o datos de ejemplo

### 2. Proponer mejoras de datos
Si tienes datos adicionales o correcciones:
1. Haz un **fork** de este repositorio
2. Crea una rama: `git checkout -b mejora/descripcion`
3. Realiza tus cambios y documenta la fuente de datos
4. Envía un **Pull Request** con una descripción clara

### 3. Mejorar documentación
Correcciones de typos, traducciones o mejoras en el README son bienvenidas.

---

## 📋 Estándares de calidad

- Los datos deben provenir de **fuentes verificables y citables**
- Incluye referencias a la fuente original (DOI, URL, institución)
- Los archivos CSV deben tener encabezados en inglés
- Los metadatos deben seguir el estándar [FAIR](https://www.go-fair.org/)
